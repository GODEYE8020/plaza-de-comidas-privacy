1. Introducción

Bienvenido a Plaza de Comidas Online. Esta Política de Privacidad explica cómo recopilamos, utilizamos, protegemos y tratamos la información personal de nuestros usuarios cuando utilizan la aplicación móvil y los servicios asociados.

3. Información que recopilamos

  • Nombre y apellidos.
  • Correo electrónico.
  • Contraseña administrada mediante un servicio seguro de autenticación.
  • Número de teléfono.
  • Dirección de entrega.
  • Ubicación GPS cuando el usuario concede el permiso.
  • Historial de pedidos.
  • Comprobantes de pago cargados por el usuario.
  • Información técnica necesaria para el funcionamiento de la aplicación.

4. Cómo utilizamos la información

  • Crear y administrar cuentas.
  • Permitir el inicio de sesión con correo electrónico y contraseña.
  • Procesar pedidos.
  • Gestionar entregas.
  • Enviar notificaciones sobre el estado de los pedidos.
  • Verificar pagos realizados mediante QR.
  • Mejorar la calidad y seguridad del servicio.

5. Pagos

Los pagos se realizan mediante código QR. El usuario carga una imagen del comprobante de pago para su verificación manual por un administrador. La aplicación no almacena números de tarjetas bancarias ni credenciales financieras.

6. Ubicación

La ubicación se utiliza únicamente para facilitar las entregas y mejorar la experiencia del usuario.

7. Cámara y archivos

La aplicación solicita acceso a la cámara o a la galería únicamente para permitir que el usuario adjunte un comprobante de pago.

8. Notificaciones

La aplicación puede enviar notificaciones relacionadas con el estado del pedido, confirmaciones y avisos importantes del servicio.

9. Seguridad

Aplicamos medidas técnicas y organizativas razonables para proteger la información personal frente a accesos no autorizados, pérdida o alteración.

10. Conservación de datos

La información se conserva únicamente durante el tiempo necesario para prestar el servicio, cumplir obligaciones legales o resolver incidencias.

11. Derechos del usuario

El usuario puede solicitar la actualización, corrección o eliminación de sus datos personales mediante el correo de contacto.

12. Eliminación de cuenta

Si un usuario desea eliminar su cuenta y los datos asociados, puede solicitarlo escribiendo a ideamakers8020@gmail.com. La solicitud será atendida conforme a la normativa aplicable y a las obligaciones legales de conservación.

13. Menores de edad

La aplicación no está dirigida intencionalmente a menores de edad sin la autorización de sus padres o tutores.

14. Cambios

Esta política puede actualizarse periódicamente. La versión publicada en esta página será la vigente.
